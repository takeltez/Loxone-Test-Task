# Test task for Loxone Electronics

### Pre-requirements
- GNU Compiler: `g++`
- GNU Make: `make`

### Makefile targets
- `all`<br>
Build project.

- `clean`<br>
Clean all utility files and directories.

- `zip`<br>
Archive all necessary files and directories.

### Usage
From root directory:<br>
`./bin/loxone`
